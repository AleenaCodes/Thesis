Web Accessiblility Project Demo

The following is a proof-of-concept demo for graphs that are interactive with the screen reader

Chart 1 - Number of users of social networks

Accessible bar graph showing number of users of social networks, select link to skip graph, or use tab to explore graph

Chart 2 - Temperature in London over a week

Accessible line graph showing temperature in London over a week, select link to skip graph, or use tab to explore graph

Chart 3 - Percentages of food groups for a balanced meal

Accessible pie chart showing percentages for a balanced meal, select link to skip graph, or use tab to explore graph
